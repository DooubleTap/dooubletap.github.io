# Contributing to DooubleTap Developer Hub

Thank you for your interest in contributing! This repository is primarily my personal developer portfolio, but contributions, suggestions, and improvements are welcome.

## How to Contribute

1. Fork the repository.
2. Create a new branch for your feature or fix:
   git checkout -b feature-name
3. Make your changes in HTML, CSS, or assets.
4. Test locally by opening index.html in a browser.
5. Commit your changes with a clear message:
   git commit -m "Add new feature / Fix typo / Update badge"
6. Push your branch to your fork:
   git push origin feature-name
7. Open a Pull Request on the main repository.

## Guidelines

- Keep the dark theme consistent.
- Only add badges that are relevant to skills or contact information.
- Respect the existing layout and typography.
- Avoid large images that slow down page load.

## Reporting Issues

Please use the Issues tab to report any bugs, typos, or suggestions for improvement.

Thank you for helping improve this portfolio!
